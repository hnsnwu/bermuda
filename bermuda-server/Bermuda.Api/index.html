Bermuda API List

Which includes the APIs of db tables.

Topic

  • GET /api/topics

    Get all topics. i.e. all

  • GET /api/topic{/id}

    Get topic by id. i.e. id=1

  • GET /api/topics{/type}

    Get topics by the type which inludes all or top.

  • GET /api/topics{/type/count}

    Get topics top list with count var. i.e. top/5

Notice

  • GET /api/notices

    Get all notices which are not solved. i.e. all

Notice Specie

  • GET /api/notice/species

    Get all notice species. i.e. all

  • GET /api/notice/specie{/id}

    Get notice specie by id. i.e. id=1

  • GET /api/notice/species/top

    Get notice species top list. i.e. top
